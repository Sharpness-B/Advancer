Designskisser                 : // - Ikke brukt https://www.figma.com/ - //

Objective                : Ødelegge planeter for materialer for å oppgradere fighter-jet så du                           kan slå vennene dine i en multiplayer racing game mode.

Controls                : Throttle (W, S), Movement Ability (SPACE), Attack Ability (MOUSE 1,                           MOUSE 2), Defensive Ability (F), Pause Menu (ESC), Radar (TAB),                           Air Roll (Q, E), Aim / Steer (MOUSEPOS.)

Kamerasystem        : 3D Cartesian Geometry som regnes ut fra, kameraobjektet

                        cam = {x: , y: ,z: ,yaw: ,pitch: ,roll: ,fov: }

Matriser                : Matriser vil bli brukt til transformasjoner, translasjoner og rotasjoner.
